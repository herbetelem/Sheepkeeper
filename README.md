# SheepKeeper - Page de téléchargement (GitHub Pages)

Cette page permet aux élèves de télécharger SheepKeeper (Android) via un lien + QR code.

## Fichiers
- `index.html` : la page
- `styles.css` : le style
- `assets/qr_sheepkeeper.png` : QR code (pointe vers le lien Drive actuel)

## Étapes GitHub Pages
1. Crée un repo GitHub (ex: `sheepkeeper`)
2. Ajoute ces fichiers à la racine du repo
3. Va dans **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main**, folder: **/(root)**
6. Attends le lien du site (ex: https://<user>.github.io/sheepkeeper/)
